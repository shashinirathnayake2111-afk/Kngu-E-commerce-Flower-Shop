## 🛠️ Tech Stack & Frameworks
 
| Technology | Purpose |
|---|---|
| **HTML5** | Semantic architecture (`<header>`, `<main>`, `<section>`, `<footer>`) for better SEO and readability |
| **Bootstrap 5** | Mobile-first responsive grid system (`row`, `col-*`) and UI components |
| **Custom CSS3** | Pixel-perfect layout adjustments, color palette, typography, and hover transitions |
| **Font Awesome v6** | Premium and clean UI iconography |
| **Google Fonts** | *Playfair Display* (headings) + *Montserrat* (body text) |
 
---
 
## 🎨 Key UI Features Implemented
 
- ✅ **Responsive Architecture** — Scales seamlessly across Mobile, Tablet, and Desktop using Bootstrap breakpoints
- ✅ **Hero Carousel Section** — Interactive slider showcasing seasonal promotional content
- ✅ **E-Commerce Product Grids** — Card components with badges (*Sale*, *New*), rating stars, and styled pricing
- ✅ **Custom Typography & Colors** — Dark Charcoal `#2c2c2c` footer + Coral Red `#ff5a50` action buttons

## 🚀 How to Run Locally
 
**1. Clone the repository:**
```bash
git clone https://github.com/shashinirathnayake2111-afk/Kngu-E-commerce-Flower-Shop.git
```
 
**2. Navigate into the project folder:**
```bash
cd Kngu-E-commerce-Flower-Shop
```
 
**3. Open in browser:**
 
Open `index.html` directly in any modern web browser, or use the **VS Code Live Server** extension for hot-reload development.
